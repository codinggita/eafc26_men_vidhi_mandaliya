# EAFC 26 Player Analytics Project

Welcome to the **EAFC 26 Player Analytics** repository. This is a full-stack project built to analyze and manage male football players from the EAFC 26 dataset.

## 📁 Repository Organization
* **[backend/](file:///c:/Users/Vidhi/Desktop/new_prj/eafc26_men_vidhi_mandaliya/backend)**: Express.js, Node.js, and MongoDB backend.
  * For setup guide, dependency listings, database seeding details, and the complete REST API endpoints reference, please refer to the **[Backend README.md](file:///c:/Users/Vidhi/Desktop/new_prj/eafc26_men_vidhi_mandaliya/backend/README.md)**.

## 🚀 Getting Started

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Set up your `.env` file (see `.env.example`).
3. Place your player dataset `EAFC26-Men.json` in `backend/src/seed/data/`.
4. Run `npm install` to install dependencies.
5. Run `npm run seed` to import players to MongoDB.
6. Run `npm run dev` to start the backend developer server.
